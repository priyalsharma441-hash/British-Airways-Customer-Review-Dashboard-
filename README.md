# ✈️ British Airways Customer Review Dashboard

## 📌 Project Overview

This project analyzes British Airways customer reviews using **Power BI** to uncover passenger satisfaction trends and provide business insights. The dashboard transforms raw review data into interactive visualizations that help identify customer preferences, service performance, and overall satisfaction.

---

## 🎯 Objectives

- Analyze overall customer satisfaction.
- Track rating trends over time.
- Identify the highest-rated aircraft.
- Compare ratings across traveller types and seat classes.
- Visualize recommendation distribution.
- Analyze review distribution by continent.
- Build an interactive dashboard with slicers for dynamic analysis.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- Data Modeling
- DAX (Basic Measures)
- CSV Datasets

---

## 📂 Dataset

The project uses two datasets:

- **ba_reviews.csv** – British Airways customer reviews
- **Countries.csv** – Country-to-continent mapping

---

## 📊 Dashboard Features

### KPI Cards
- Average Rating
- Average Seat Comfort
- Average Entertainment
- Average Food & Beverages
- Average Cabin Staff Service
- Average Ground Service
- Average Value for Money

### Visualizations
- Average Rating by Year
- Top Rated Aircraft
- Ratings by Traveller Type
- Ratings by Seat Type
- Recommendation Distribution
- Reviews by Continent

### Interactive Slicers
- Seat Type
- Traveller Type
- Trip Verified
- Continent

---

## 🔍 Key Insights

- Overall average customer rating is **4.19 / 10**.
- Europe contributes the highest number of customer reviews.
- Solo Leisure travellers tend to give higher ratings than other traveller types.
- Economy Class receives the largest share of reviews.
- Aircraft performance varies, with some aircraft receiving consistently higher ratings.
- More than half of reviewers recommended British Airways.
